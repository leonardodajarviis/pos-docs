# Tài liệu Hệ thống POS

Dự án này chứa tài liệu chi tiết về các chức năng của hệ thống Point of Sale (POS).

## Xem tài liệu

Truy cập [GitHub Pages](https://jarviisha.github.io/pos-docs/) để xem tài liệu đầy đủ.

## Cấu trúc dự án

```
pos-docs/
├── BANG-GIA/              # Module Bảng Giá
├── CHINH-SACH-GIA-BAN/    # Module Chính Sách Giá Bán
├── CHINH-SACH-GIA-MUA/    # Module Chính Sách Giá Mua
├── MA-GIA/                # Module Mã Giá
└── Templates              # Các mẫu tài liệu
```

## Modules

- **Bảng Giá**: Quản lý bảng giá sản phẩm/dịch vụ
- **Chính Sách Giá Bán**: Quản lý các chính sách giá bán
- **Chính Sách Giá Mua**: Quản lý các chính sách giá mua
- **Mã Giá**: Quản lý mã giá trong hệ thống
